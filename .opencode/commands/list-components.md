description: List React component files, optionally within a component subdirectory
---

Load the `list-components` skill and follow it exactly.

Optional component scope supplied by the user:

`$ARGUMENTS`

If the argument is empty or whitespace, inventory all discovered component roots. Otherwise, use the full argument as the requested repository-relative component scope.

This command is read-only. Return only the requested numbered inventory and its summary, or the exact no-results message required by the skill.
