## Filebase S3 API Configuration

From the Filebase docs:
- **Endpoint**: `https://s3.filebase.io`
- **Region**: `auto`
- **Signature Version**: `v4` (AWS4-HMAC-SHA256 only, no SigV2)
- **Protocol**: HTTPS-only on port 443
- **Rate Limit**: 500 requests per second
- **Compatible Tools**: AWS CLI, AWS SDKs, boto3, S3 Tools
- **Addressing Styles**: Both path style and virtual-host style supported
