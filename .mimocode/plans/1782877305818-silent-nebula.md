# Fix Vercel Build: `updatePassword` Type Error

## Root Cause

The Vercel build fails with:

```
Type error: Property 'updatePassword' does not exist on type 'InferAPI<...>'
./src/app/api/profile/change-password/route.ts:33:20
```

The method name is wrong. **better-auth exposes `changePassword`, not `updatePassword`.** The `changePassword` method is a core better-auth endpoint (no plugin needed) and is available in the installed version `1.6.23`.

## Prisma Check

**No missing migrations.** The single init migration (`20260626114754_init`) matches the current schema exactly — all 10 models, all columns, indexes, unique constraints, and foreign keys are in sync.

## Fix

**File:** `src/app/api/profile/change-password/route.ts` (line 33)

```diff
-    await auth.api.updatePassword({
+    await auth.api.changePassword({
```

Single-line change. The `changePassword` method accepts the same `body: { currentPassword, newPassword }` shape that the route already passes, so no other changes are needed.
