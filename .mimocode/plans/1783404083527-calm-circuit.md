~40 lines removed per form.

**File:** `src/components/auth/auth-form-field.tsx`

Wraps label + input + error message. Accepts `htmlFor`, `label`, `error`, `children`.

~30 lines removed per form.

### Step 3: Create `AuthFormDivider` component
**File:** `src/components/auth/auth-form-divider.tsx`

"Or continue with" divider + GitHub button. Accepts `callbackURL`.

~12 lines removed per form.

### Step 4: Refactor both forms
