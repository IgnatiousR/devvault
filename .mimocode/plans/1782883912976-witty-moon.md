# Dynamic Keyboard Shortcut Display

## Task
Detect user's OS and show ⌘K for Mac, Ctrl+K for others in the search placeholder.

## File to Modify
- `src/components/dashboard/top-bar.tsx`

## Implementation
1. Add `useState` and `useEffect` to detect platform
2. Use `navigator.userAgent` to check for Mac
3. Show `(⌘K)` for Mac, `(Ctrl+K)` for others
4. Default to `(Ctrl+K)` during SSR to avoid hydration mismatch
